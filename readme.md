## [隨機句子產生器 1.0.0](https://random-word0412.herokuapp.com/)
### 說明:
網站的資料庫充滿了不同的名詞 動詞 形容詞
會隨機組合成一個句子
![](https://i.imgur.com/9FkP7fQ.png)

使用者能資料庫自行添加想要的詞彙
讓下一個人組合句子的時候加到你創造的詞


![](https://i.imgur.com/0oMXTQP.png)

其實玩法很簡單 試看看就知道了
